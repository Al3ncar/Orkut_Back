# 📘 API Orkut

API REST simples para gerenciamento de usuários e posts.

## 🚀 Base URL
[https://api-orkut-back-two.onrender.com](https://api-orkut-back-two.onrender.com)



## 📄 Documentação completa

Acesse a documentação detalhada da API no Postman:

🔗 https://documenter.getpostman.com/view/52849281/2sBXqDtPT1



## 👤 Rotas de Usuário

- **GET /user** → Lista usuários  
- **POST /user** → Cria um usuário  
- **DELETE /user/:id** → Deleta um usuário  



## 🔐 Autenticação

- **POST /login** → Retorna um token JWT  



## 📝 Rotas de Posts

- **GET /posts** → Lista posts  
- **POST /post** → Cria um post  
- **PUT /post/:id** → Atualiza um post  
- **DELETE /posts/:id** → Deleta um post  



## 🛠️ Tecnologias

- Node.js  
- Express  
- JWT  
- Bcrypt  



## 📌 Observações

- Senhas são criptografadas  
- Autenticação via token  
- API simples para fins de estudo
